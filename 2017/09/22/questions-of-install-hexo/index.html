questions of install hexo

2017-09-22
hexo

重启电脑后hexo 命令找不到：

解决方法：
linux的执行命令都放在 /usr/local/bin

使用 whereis 命令找到npm 的安装位置，找打hexo 使用ln -s 源文件 目标文件 设置软链接 到 usr/local/bin 文件中，就能找到命令了。
